To execute this program:

1. all dependencies must first be installed (pandas, tensorflow, etc.)
2. unzip "stock data.zip" and move the contents to a desired path
3. root path in "definitions.py" must be modified to this path

Other notes:
- There is additional code that has been commented out to convert the resulting CSV file to an .arff file if the user wishes to use this data with Weka

- There is additional code that has been commented out to allow the user to download excel files from yahoo

- Individual Reports generates an excel spreadsheet with a number of attributes about the ticker that the user inputs
